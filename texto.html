Texto

 

Son las palabras que utilizamos para transmitir información. Pueden ser títulos, párrafos, etiquetas, etc.

Función: Transmitir información, crear jerarquías, generar emociones y guiar al usuario.
Uso: Títulos, subtítulos, párrafos, etiquetas, botones, menús.
Ejemplos: El texto en una página web, las instrucciones de un videojuego, los créditos de una película.

Formatos mas utilizados

.txt: el formato estándar para archivos de texto sin formato. Se basa principalmente en el sistema de codificación ASCII y solo contiene texto sin formato.
.doc: el formato principal del programa Microsoft Word hasta 2007. Puede contener texto formateado, así como imágenes y gráficos.
.docx: mayor desarrollo del formato DOC y hoy en día el formato estándar para archivos de Word. Basado en XML y se considera que ahorra más espacio que los archivos DOC.
.rtf: formato de archivo de Microsoft que se suele utilizar como formato de intercambio entre varios programas de procesamiento de textos y sistemas operativos.
.pdf: formato de archivo independiente de la plataforma desarrollado por Adobe. Es particularmente adecuado para enviar y colaborar en documentos.
.odt : formato de archivo de texto de la categoría OpenDocument Format (ODF). Normalmente se crea utilizando programas gratuitos como Apache OpenOffice Writer.
.csv: formato de archivo que se utiliza para intercambiar y almacenar datos estructurados de forma sencilla. Característica especial: la información está separada por comas en lugar de almacenarse en columnas.
.xml: formato de archivo de texto relativamente simple basado en el lenguaje de marcado XML. Comúnmente utilizado en programación y desarrollo web.
.log: tipo de archivo utilizado para el almacenamiento y registro automático de eventos, procesos y transacciones.
