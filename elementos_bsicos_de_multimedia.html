Elementos Básicos de Multimedia

 

Imagina que estás creando una historia. Puedes usar palabras (texto), dibujos (imágenes), sonidos y hasta videos para hacerla más interesante. El diseño multimedia es como construir esa historia, pero utilizando herramientas digitales. Combina diferentes elementos para crear experiencias visuales y auditivas que sean atractivas y fáciles de entender.
